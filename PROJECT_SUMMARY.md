## 🎯 Core Algorithms Implemented

### 1. Cycle Phase Calculation
- Menstrual: Days 1-5
- Follicular: Days 6-14
- Ovulation: Days 15-17
- Luteal: Days 18-28

### 2. Fertility Window Detection
- Window: Day 14 ± 5 days
- Peak fertility: Day 14-15
- Pregnancy probability calculation

### 3. Ovulation Prediction
- Standard: Day 14 of cycle
- Adjustable based on history

### 4. Cycle Statistics
- Average cycle length (21-35 days)
- Cycle variability (standard deviation)
- Irregularity detection
- Trend analysis

### 5. Wellness Scoring
- Phase-based wellness score (1-10)
- Daily mood aggregation
- Trend visualization
